В данном проекте использовано машинное обучение. Спрогнозирована вероятность оттока (на уровне следующего месяца) для каждого клиента, выявлены признаки в большей степени связанные с оттоком; сформированы типичные портреты пользователей: выделены наиболее яркие группы, охарактеризованы их основные свойства групп, предложены мероприятия для снижения оттока на основании проведенного анализа
